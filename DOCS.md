# Homebrew Tap Documentation

## Add the tap

```sh
brew tap shroudb/tap
```

## Available formulae

| Formula             | Description                                                     |
|---------------------|-----------------------------------------------------------------|
| `shroudb`           | Encrypted key-value database with RESP3 protocol                |
| `shroudb-cipher`    | Encryption-as-a-service with key lifecycle management           |
| `shroudb-sigil`     | Credential envelope engine with encrypted PII storage           |
| `shroudb-forge`     | Internal Certificate Authority with OCSP                        |
| `shroudb-chronicle` | Tamper-evident audit log with hash-chained events               |
| `shroudb-sentry`    | Policy-as-code authorization engine with signed JWT decisions   |
| `shroudb-courier`   | Secure notification delivery (SMTP, webhook, WebSocket)         |
| `shroudb-keep`      | Secrets manager with path-based access control and versioning   |
| `shroudb-veil`      | Encrypted search with blind indexing over E2EE data             |
| `shroudb-stash`     | S3-backed encrypted blob storage with crypto-shred              |
| `shroudb-scroll`    | Durable append-only event log with cursored readers and groups  |
| `shroudb-moat`      | Unified single binary embedding all ShrouDB engines             |

## Install

```sh
brew install shroudb/tap/shroudb
brew install shroudb/tap/shroudb-cipher
brew install shroudb/tap/shroudb-sigil
brew install shroudb/tap/shroudb-forge
brew install shroudb/tap/shroudb-chronicle
brew install shroudb/tap/shroudb-sentry
brew install shroudb/tap/shroudb-courier
brew install shroudb/tap/shroudb-keep
brew install shroudb/tap/shroudb-veil
brew install shroudb/tap/shroudb-stash
brew install shroudb/tap/shroudb-scroll
brew install shroudb/tap/shroudb-moat
```

## Upgrade

```sh
brew upgrade shroudb/tap/shroudb
```

## Uninstall

```sh
brew uninstall shroudb
```
